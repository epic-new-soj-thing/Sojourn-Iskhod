## About The Pull Request
<details>
<summary>About The Pull Request</summary>
<hr>

Added descriptions for some admin verbs (with more to come when I figure out what they do, those have #TODO-ISKHOD notes next to them), removed unneeded whitespace, changed inconsistant order of "set name", "set category" and "set desc", edited some comments and reorganised the verb order to be based on categories (this hopefully should not effect how it looks in-game as I did keep the order the same per category). It compiles but I haven't tested if the reorganisation has broken anything, so all admin verbs will have to be clicked on to be tested.
